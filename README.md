# [Waline](https://github.com/walinejs/aliyun-fc-starter)


Waline 是一款基于 Valine 衍生的简洁、安全的评论系统，该项目是使用[阿里云函数计算](https://fc.console.aliyun.com/)部署的一个 [Waline](https://waline.js.org) 应用示例。
